## The 10 Commandments (One-Line Version)

1. **Radical Clarity**: Say it simpler
2. **Data Over Adjectives**: Show numbers, not descriptions
3. **Honesty First**: Acknowledge failures upfront
4. **First-Principles**: Explain from fundamentals
5. **Economy**: Cut every wasted word
6. **Active Voice**: People do things
7. **Context**: Always explain "why"
8. **Attribution**: Credit sources
9. **Forward-Looking**: Point to next, drive outcomes
10. **Format Function**: Structure serves message

## Banned vs. Golden

### ‚ùå Banned Words/Phrases

**Vague Adjectives**:
- Significant, substantial, considerable
- Very, really, quite, actually

**Corporate Jargon**:
- Leverage, synergy, paradigm
- Going forward, moving forward
- Drive toward, laser-focused, de-risk
- Socialize, alignment, stakeholder engagement
- Stand up, operationalize, at scale
- Touch base, circle back, shift right

**Euphemisms**:
- Challenges (say "problems")
- Opportunities for improvement (say "failures")
- Rightsizing (say "layoffs")

### ‚úÖ Golden Phrases

**Transitions**:
- "What happened?"
- "This resulted in..."
- "What does this mean for..."
- "Put more simply, X > Y"
- "The reason lies with..."

**Causal**:
- "Because [reason], [action]"
- "Since [context], [implication]"

**Analysis**:
- "Conventionally, [X]... However, [Y]"
- "Throughout [time period], [observation]"

**Directives**:
- "Run it by [name] before you send it pls"
- "You need to..."

## Universal Editing Checklist

Use for ALL formats:

### Content
- [ ] Specific numbers, not adjectives?
- [ ] Failures acknowledged upfront?
- [ ] First-person accountability ("I/we")?
- [ ] Explained "why" for every "what"?
- [ ] Active voice (not passive)?
- [ ] Sources attributed?

### Language
- [ ] Could cut 20% more words?
- [ ] No banned phrases?
- [ ] No hedging ("possibly," "maybe")?
- [ ] Technical terms defined on first use?
- [ ] Reading level ~10th grade?

### Structure
- [ ] Most important information first?
- [ ] Headers signal topic shifts?
- [ ] Bullets for parallel items, numbers for sequential?
- [ ] Tables for comparative data?

### Next Steps
- [ ] Forward-looking (what happens next)?
- [ ] Clear ownership assigned?
- [ ] Specific timelines/dates?

## Format-Specific Checklists

### Customer Brief
- [ ] **Opening**: Table first (Pattern 2) OR Strategic hook (Pattern 1)?
- [ ] **Losses**: Separate section, no excuses?
- [ ] **Roadmap**: Specific deliverables + dates?
- [ ] **Asks**: Numbered with clear ownership?
- [ ] **Status**: üü©üü®üü• indicators used?

### Email
- [ ] **Length**: Under 200 words?
- [ ] **Purpose**: Single topic only?
- [ ] **Greeting**: Minimal or none?
- [ ] **Closing**: Name or nothing?
- [ ] **Formatting**: Plain text (no bold/italics)?

### Annual Letter
- [ ] **Performance**: Table before narrative?
- [ ] **Themes**: 3-5 major sections?
- [ ] **Data**: Charts with titles, labels, sources?
- [ ] **Closing**: Personal reflection + signature wisdom?

### Policy Ideas
- [ ] **Verbs**: Imperative (Build, Create, Mandate)?
- [ ] **Structure**: 4-part (Problem ‚Üí Solution ‚Üí Implementation ‚Üí Impact)?
- [ ] **Bullets**: 70-80% of content?
- [ ] **Impacts**: Categorized by dimension?

### Learn with Me
- [ ] **Slides 1-3**: Title, "How to Read", Table of Contents?
- [ ] **One Idea**: Strictly one per slide?
- [ ] **Citations**: On every data slide?
- [ ] **Final Slide**: Legal disclaimers?

## Decision Trees

### "Should I use bullets or prose?"

```
Is this a Policy Ideas Brief?
  YES ‚Üí Use bullets (70-80% bullets)
  NO ‚Üì

Is this a list of parallel items or action steps?
  YES ‚Üí Use bullets
  NO ‚Üì

Is this explaining causal relationships or building an argument?
  YES ‚Üí Use narrative prose
  NO ‚Üì

DEFAULT ‚Üí Use narrative prose
```

### "Pattern 1 or Pattern 2 for Customer Brief?"

```
Multiple projects needing comparison?
  YES ‚Üì

Reader already has context?
  YES ‚Üì

Time-constrained (< 5 min)?
  YES ‚Üí Pattern 2 (Decision-Driven: table-first)
  NO ‚Üí Pattern 1 (Strategic Hook: narrative-first)
```

### "Should I cut more words?"

```
After first draft: Cut 50%
After second draft: Cut 30% more
After third draft: If uncertain, cut 20% more

Target reductions:
- Corporate jargon ‚Üí 240 words to 47 words (80%)
- Email ‚Üí Always under 200 words
- Customer brief ‚Üí 50% reduction typical
```

## Formatting Quick Rules

| Element | When to Use | When NOT to Use |
|---------|-------------|-----------------|
| **Bold** | Metrics, key findings, headers | Every sentence for emphasis |
| *Italics* | Subsection leads, quote emphasis | Casual emphasis |
| Bullets | Parallel non-sequential items | Sequential steps (use numbers) |
| Numbers | Sequential steps, priorities | Parallel items (use bullets) |
| Tables | Comparing across 2+ dimensions | Simple lists |
| Charts | Trends/relationships over time | When table is clearer |

## Common Mistakes & Fixes

| Mistake | Fix |
|---------|-----|
| "We saw significant growth" | "Revenue grew 47% QoQ from $2.1M to $3.1M" |
| "Mistakes were made" | "We made mistakes. I learned from them." |
| "Despite some challenges..." | State the challenge first with data, then wins |
| Email > 200 words | Split into multiple emails, one per topic |
| "Over the next 100 days" | Use specific days: "Over the next 80 days" |
| Missing "why" | Add: "Because [reason], [action]" |
| Passive voice | "We did X" not "X was done" |
| Table without context | Add: "Calculated using [method] assuming [assumptions]" |

## Metric Presentation Patterns

### Dual Format (Percentage + Dollar)
```
This 30% drop-off (~$3M/yr) of lost potential revenue
```

### Before/After Comparison
```
Revenue grew 47% QoQ from $2.1M to $3.1M
```

### Range with Context
```
Total ROI: $2.4M/yr in cost (upper end) ‚Üí $20.7M/yr in upside (low end)
```

### Conservative Estimate
```
Lost revenue opportunity to well north of $250M/yr
```

## Header Hierarchy

```
# Title (only one H1)
## Major Section
### Subsection
```

**Maximum nesting**: 3 levels
**Never skip levels**: Don't jump from H1 to H3

## The 30-Second Test

Before sending any communication, ask:

1. **Purpose**: Can I state the single purpose in one sentence?
2. **Data**: Did I replace adjectives with numbers?
3. **Honesty**: Did I acknowledge failures first?
4. **Brevity**: Could I cut 20% more without losing meaning?
5. **Action**: Is it clear what happens next?

If any answer is "NO," revise before sending.
