# CP Writing Style Guide v2.1.0

This skill provides comprehensive guidance on writing in Chamath Palihapitiya's distinctive communication style.

## When to Use This Skill

Claude should invoke this skill when:
- User asks for help writing any type of document
- User requests document review or editing
- User asks "how should I write..." or "what format should I use..."
- User mentions any of the 5 formats: Annual Letters, Customer Briefs, Emails, Policy Ideas Briefs, or Learn with Me Presentations
- User asks about writing style, tone, structure, or formatting
- User needs templates or examples for business communications

## What This Skill Provides

### 10 Universal Principles
1. **Radical Clarity**: Say it simpler
2. **Data Over Adjectives**: Show numbers, not descriptions
3. **Honesty First**: Acknowledge failures upfront
4. **First-Principles Thinking**: Explain from fundamentals
5. **Economy of Language**: Cut every wasted word
6. **Active Voice**: People do things
7. **Context Provision**: Always explain "why"
8. **Attribution**: Credit sources
9. **Forward-Looking & Action-Oriented**: Point to next and drive outcomes
10. **Format Follows Function**: Structure serves message

### 5 Complete Format Playbooks

1. **Annual Letters** (2,000-5,000 words)
   - Strategic reflection + performance reporting
   - For investors/public audience
   - Includes personal philosophy + data + portfolio updates

2. **Customer Briefs** (500-2,000 words)
   - Progress reporting + stakeholder management
   - Internal and public variants
   - Heavy use of tables for accountability
   - Includes Progress Update Briefs sub-format (Section 3.6)

3. **Emails** (50-200 words)
   - Quick tactical communication
   - 3 types: Philosophy sharing, Personnel changes, Directives
   - Minimal formatting, maximum clarity

4. **Policy Ideas Briefs** (3,000-5,000 words)
   - Policy recommendations + implementation plans
   - Uses imperative commands (Build, Create, Mandate)
   - Numbered hierarchy with 4-part structure per proposal

5. **Learn with Me Presentations** (40-80 slides)
   - Educational deep-dives on complex topics
   - Visual-first format with minimal text per slide
   - Standardized structure: Title, How to Read, TOC, Chapters, Disclaimer

## Key Decision Framework

```
Purpose: Quick tactical → Email (50-200 words)
Purpose: Progress update → Progress Update Brief (300-500 words summary + appendix)
Purpose: Multi-project proposal → Customer Brief (500-2,000 words)
Purpose: Strategic reflection → Annual Letter (2,000-5,000 words)
Purpose: Policy recommendations → Policy Ideas Brief (3,000-5,000 words)
Purpose: Educational deep-dive → Learn with Me (40-80 slides)
```
